# Completed Execution Plans

Archive completed execution plans here when they remain useful as historical context.

Keep each completed plan concise:

- What changed.
- Why it changed.
- Important decisions.
- Verification performed.
- Follow-up tech debt, if any.

Do not store run logs, screenshots, generated assets, or machine-specific paths here. Use `.codex-temp/` for local scratch files and keep it uncommitted.
